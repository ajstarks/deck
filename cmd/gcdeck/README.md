# fcdeck

![gcdeck](gcdeck.png)

## Options


```
fcdeck [options] file

Options:

  -page int
    	initial page (default 1)
  -pagesize string
    	pagesize: w,h, or one of: Letter, Legal, Tabloid, A3, A4, A5, ArchA, 4R, Index, Widescreen (default "Letter")
  -sans string
    	default font
  -title string
    	slide title
```
