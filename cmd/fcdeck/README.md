# fcdeck

![fcdeck](fcdeck.png)

## Options

```
fcdeck [options] file

Options:
  -grid float
        grid percent (default 10)
  -pagesize string
        pagesize: w,h, or one of: Letter, Legal, Tabloid, A3, A4, A5, ArchA, 4R, Index, Widescreen (default "Letter")
  -sans string
        default font
  -title string
        slide title
```
